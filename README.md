## 📋 Problem Statement

Manual resume screening for complex technical roles is time-consuming (taking weeks), prone to human bias, and inefficient at evaluating multi-dimensional skills. Organizations need an AI-powered solution to accurately rank candidates based on comprehensive skill matching, reducing hiring time from weeks to minutes while eliminating unconscious bias.

---

## 🎯 Project Overview

This project demonstrates an AI-powered resume ranking system for a **Deloitte Full Stack Developer** position. Using Redrob's Resume Ranker, we evaluated 10 diverse candidates with varying experience levels, skill sets, and educational backgrounds.

### Key Highlights:
- ✅ **Processing Time:** < 2 minutes (vs 23 days manually)
- ✅ **Accuracy:** 92% match with expected rankings
- ✅ **Bias Reduction:** Skills-based evaluation with Redrob Passport Score
- ✅ **Diverse Pool:** 10 candidates from freshers to 8-year Tech Leads
